03  Comment je construis

  • Front / app : Next.js 16 (App Router), React 19, TypeScript strict, Tailwind, shadcn/ui, Vercel.
  • Data / auth : Supabase (Postgres, RLS, pgvector), Stripe, Drizzle/Prisma selon le projet.
  • IA : Claude (Haiku classification, Sonnet drafts, Opus raisonnement), Mistral pour l'OCR et la souveraineté EU, Vercel AI Gateway, embeddings Voyage AI, RAG sur pgvector, tool-use agentique.
  • Terrain : Pappers, BODACC, JORF, DVF Etalab pour les données réelles. Je pars toujours d'un cas client concret.
